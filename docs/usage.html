Using DnDnD

DnDnD has two surfaces: the DM web dashboard for running the campaign, and player slash commands in Discord for taking actions. This guide tours the dashboard, walks a typical session, and lists every player command.

A typical session (DM)

  1. In Discord, run /setup once to create the campaign's channel structure (needs Manage Channels).
  2. In the dashboard, create or select a Campaign (Campaigns).
  3. Have players join with /register, then approve them in Character Approvals.
  4. Build a battle map in the Map Editor, or import a full-art map from Tiled (see the Tiled guide).
  5. Assemble the fight in the Encounter Builder — pick the map, place PCs and SRD monsters.
  6. Go live and run the fight from the Combat Manager (initiative, HP, conditions), using the Turn Builder to compose turns.
  7. Resolve player whispers, skill-check narration, and undo requests from the DM Queue; post story beats from Narrate.

A health banner at the top always shows the bot's Discord connection status.

Getting started (players)

Players act entirely from Discord slash commands. To get a character:

  • /register — claim a DM-made character, build one, or import (run with no name to choose).
  • /import ddb-url — import a character from D&D Beyond.
  • /create-character — build a character in the web portal.
  • /character / /status — view your sheet summary and current status.

💡Run /help any time for the full in-Discord command list, or /help topic for a specific area.

Player command reference

Arguments are shown as required and optional. All commands below are player-facing; only /setup DM is permission-gated (Manage Channels).

Movement

/moveMove to a coordinate on the battle map. coordinate
/flyFly to a given altitude in feet. altitude
/distanceMeasure distance between targets. target target2

Combat

/attackAttack a target with a weapon. target weapon gwm sharpshooter reckless offhand thrown …
/castCast a spell, with metamagic & slot options. spell target level ritual …
/bonusUse a bonus action. action args
/actionUse an action on your turn. action args spell slot
/shoveShove a target (push, prone, or grapple). target mode
/interactInteract with an object. description
/reactionDeclare, cancel, or clear reactions for the round.
/commandCommand a companion creature. creature_id action target
/deathsaveRoll a death saving throw.
/doneEnd your turn.

Checks, saves & rest

/checkMake an ability or skill check. skill adv disadv target dc
/saveMake a saving throw. ability
/restTake a short or long rest. type

Status & inventory

/statusShow your character's current status.
/inventoryShow your inventory.
/equipEquip an item. item offhand armor
/useUse a consumable item. item
/giveGive an item to another creature. item target
/lootLoot the area or a defeated creature.
/attune / /unattuneBegin or end attunement with a magic item. item
/preparePrepare your spell list for the day. spells class subclass
/undoRequest to undo your last action. reason

Communication

/whisperSend a private message to the DM. message

Character management

/registerGet a character — claim, build, or import. name
/importImport a character from D&D Beyond. ddb-url
/create-characterBuild a character in the web portal.
/characterShow your character sheet summary.
/recapShow a recap of recent combat rounds. rounds
/retireRetire your character from the campaign. reason

Utility

/helpShow the full command list. topic
/setup DMCreate the full channel structure for this campaign (Manage Channels).
